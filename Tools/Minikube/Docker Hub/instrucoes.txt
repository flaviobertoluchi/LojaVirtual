________________________________________________________________________________________________________

1. Iniciar o Minikube.

________________________________________________________________________________________________________

2. Baixar o arquivo lojavirtual.yaml:
https://github.com/flaviobertoluchi/LojaVirtual/blob/master/Tools/Minikube/Docker%20Hub/lojavirtual.yaml

________________________________________________________________________________________________________

3. Deploy:

kubectl apply -f lojavirtual.yaml

________________________________________________________________________________________________________

4. Encaminhando a porta do host para o nginx (o terminal fica preso):

kubectl port-forward -n lojavirtual service/lojavirtualnginx 80:80

________________________________________________________________________________________________________

5. Acessar a aplicação em:

http://localhost

________________________________________________________________________________________________________
